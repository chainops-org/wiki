# Регистрация

После успешного запуска узла SSV оператор должен зарегистрировать его в смарт-контракте SSV, чтобы его можно было обнаружить с помощью валидаторов.

Вы можете зарегистрировать своего оператора через <a href="https://app.prater.ssv.network/">веб-приложение</a> SSV network или с помощью транзакции непосредственно в смарт-контракте

При регистрации вашего оператора вам необходимо будет указать следующие параметры:


* Имя оператора - Не редактируется после завершения транзакции (только английские символы)
* Открытый ключ оператора - генерируется в процессе установки узла (формат base64)
* Запрашиваемая плата - Оплачивается обслуживаемыми валидаторами
